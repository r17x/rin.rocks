#### Graphql Query menggunakan `graphql-tag`
Untuk kalian pengguna `Javascript`, kita dapat menuliskan _query graphql_ menggunakan pustaka `graphql-tag`. Hal yang paling umum dan sering kita lakukan pertama kali yakni `import`, setelah menambahkan pustaka tersebut.
   
Disini, kita meng-_import_ `default` dengan memberikan penamaan `gql`.


```js  query.js
import gql from 'graphql-tag'
```
---


Maka, kita dapat mendeklarasikan sebuah `variable` yang berisikan `query graphql` dengan menggunakan _Sintaksis_ dari `graphql-tag`, seperti _Code_ diberikut.


```js query.js 3:13
import gql from 'graphql-tag'

const GET_ALL_POKEMONS = gql`
 // pasang query mu disini. 
`
```
